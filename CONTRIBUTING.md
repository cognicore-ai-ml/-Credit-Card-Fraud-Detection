# Contributing Guidelines

Thank you for your interest in contributing to Credit Card Fraud Detection!

## 🧩 How to Contribute
1. Fork the repository.
2. Create a new branch for your feature or fix:
   ```bash
   git checkout -b feature-name
   ```
3. Make your changes and test them locally.
4. Commit your changes:
   ```bash
   git commit -m "Add new feature or fix"
   ```
5. Push to your branch and open a Pull Request.

## 🧠 Code Style
- Follow PEP 8 guidelines for Python.
- Use clear variable names and concise comments.
- Keep functions modular and reusable.

## 🧪 Testing
Before submitting, ensure your code runs without errors:
```bash
python src/train_model.py
python src/evaluate.py
```

## 💬 Feedback
If you find bugs or have suggestions, open an issue in the repository.
